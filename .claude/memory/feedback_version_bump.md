バージョン番号の変更は **西村さんが指示したときのみ** 行う。

**Why:** インストーラー生成を頼まれるたびに番号を上げる癖が v1.03 以降繰り返された。リリース時点での番号が不自然に大きくなる懸念を西村さんが持っている。v1.09 時点で指摘を受けた。

**How to apply:**
- 「インストーラーを作って」と言われても、バージョン番号はそのまま維持する
- バージョンを上げる場合は「v1.0X に上げてよいですか？」と必ず確認してから変える
- 前のバージョンがテスト済みであっても、西村さんの指示なしに番号を進めない
