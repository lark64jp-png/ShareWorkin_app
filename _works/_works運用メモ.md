# `_works` 運用メモ

`H:\ShareWorkin_app\_works` は、利用者が必要に応じて commit し、ChatGPT / Claude / Codex との相談やローカル確認に使う交流点として扱う。

## 基本方針

- `_works` は通常のコード同期では自動的な commit / push 対象に含めない
- Git/GitHub は、コード・設定・文書など再現可能な管理資産の同期に集中させる
- `exe` `zip` などの配布成果物は Git 管理対象に含めない
- 成果物まで Git 同期対象にすると、成果物同期という別責務と追加確認が必要になるため分離を維持する

## 使いどころ

- AI との相談メモ
- ローカル確認結果の控え
- 運用判断の一時整理
- 必要時のみ利用者判断で commit する補助資料
